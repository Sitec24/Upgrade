5.0.200 Release Notes

  • Integration with Urovo's portable data terminal (PDT) scanners has been added.
  • Support for working with the ASCII 29 (GS) delimiter character in "keyboard break" mode has been implemented.
  • When displaying tables on the PDT, automatic width selection is implemented.
  • Fixed an error related to setting the color for the active input field. Previously, the configuration specified in the "PDT Form Settings" directory was not applied.
  • Fixed other errors.
