5.0.206 Release Notes

  • Integration with data collection terminal scanners from Meferi and iData has been added.
  • Added support for entering a two-digit year when inputting dates. Fixed an issue in the integration with Zebra portable data terminals.
  • Fixed an error where, on certain PDTs, the configured automatic input would not trigger after scanning a barcode containing the separator character ASCII 29 (GS) in "keyboard wedge break" mode.
  • Fixed an issue where the mobile client could freeze during data loading and subsequently display a network connection error. This occurred in inventory workflows when master container tracking was enabled.
  • Fixed other errors.
