5.0.191 Release Notes

  • The possibility of selecting the mobile client settings in the context of the user profile is realized.
  • Fixed the identified errors.

5.0.190 Release Notes

  • New setting: “Retention period”. It specifies the retention period of the event log. The period is set in days, with values from 1 to 9 available.
  • Now you can send messages from the mobile client to a PDT, to the system administrator. Sending messages is available for 1C:WMS 5.1.2.11 or later.
  • Changed the display of server availability check and added database and licensing server availability check (1C:CLS). When available, a green check mark appears next to each check object.
  • Now you can view product images when performing operations. An image will be opened if it is set for a product in 1C:WMS. Transferring images to a PDT is available for 1C:WMS 5.1.2.11 or later.
  • Fixed the identified errors.
